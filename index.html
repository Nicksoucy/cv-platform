Questions fréquentes

Mes informations sont-elles envoyées sur internet ?

Non. Tout le contenu reste dans le navigateur de votre appareil (localStorage). La seule connexion externe est vers l'API Anthropic si vous activez l'assistant IA avec votre propre clé.

Le PDF est-il lisible par les systèmes ATS ?

Oui. Le PDF est généré par le moteur d'impression du navigateur, donc le texte est sélectionnable et analysable par les logiciels de tri des candidatures.

Puis-je avoir plusieurs CV ?

Oui — créez autant de CV que vous voulez (par exemple un pour le commercial, un pour l'événementiel). Le sélecteur en haut de la page vous permet de basculer entre eux.

Comment l'assistant IA fonctionne-t-il ?

Vous collez votre propre clé API Anthropic dans les réglages IA. Les requêtes vont directement à l'API, sans serveur intermédiaire. Pour un usage en équipe, il faudra ajouter un proxy serveur.
